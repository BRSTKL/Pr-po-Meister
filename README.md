# Pr-po-Meister 🇩🇪

**Pr-po-Meister**, Almanca fiillerin aldıkları edatları (Verben mit Präpositionen) ve edat zamirlerini (Präpositionaladverbien - `da(r)-` / `wo(r)-`) interaktif bir şekilde öğrenmenizi sağlayan modern ve duyarlı bir web uygulamasıdır.

## 🚀 Özellikler

* **Geniş Kelime Havuzu (A1-B2):** A1 seviyesinden B2 seviyesine kadar olan 170+'den fazla fiil ve edat kombinasyonu.
* **Akıllı Örnekleme (Weighted Sampling):** Sınavlarda zayıf olduğunuz, yanlış yaptığınız kelimelere öncelik veren akıllı algoritma.
* **Gelişmiş PA Quiz Modu:** Fiillere uygun `da-`/`dar-` ve `wo-`/`wor-` edat zamirlerini aktif hatırlama ile test eden özel quiz alanı.
* **Dinamik Eşleme & İpuçları:** Cümlelerdeki edat bükümlerini (`zum`, `wovon` vb.) otomatik maskeleyerek ipucu vermeden gerçek bilginizi ölçen altyapı.
* **Sesli Telaffuz (TTS):** Alman seslendirme motoruyla entegre telaffuz dinleme desteği.
* **Favoriler:** Zorlandığınız kelimeleri favorilere ekleyip sadece onlardan oluşan özel testler oluşturma imkanı.

## 🛠️ Teknolojiler

* HTML5 (Semantik Yapı)
* Vanilla CSS (Modern Karanlık Tema & Responsive Tasarım)
* Vanilla JavaScript (Dinamik Durum Yönetimi & Regex Tabanlı Maskeleme)
* LocalStorage (Öğrenme İlerlemesi ve Favorilerin Cihazınızda Saklanması)
